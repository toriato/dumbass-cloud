기타
  • 업스케일 모델 
    • https://upscale.wiki/wiki/Model_Database
  • 임베딩 (Textual Inversion = TI)
    • 프롬프트에 꺽쇠(<) 입력하면 사용 가능한 임베딩 목록 나옴
    • https://cyberes.github.io/stable-diffusion-textual-inversion-models
    • https://huggingface.co/sd-concepts-library
    • https://huggingface.co/Nerfgun3
  • 기능 및 스크립트
    • (extension) Aesthetic Gradients
    • (extension) Dynamic Prompts
    • (extension) Booru tag autocompletion for A1111
    • (extension) Tokenizer
    • (script) txt2palette
    • (script) StylePile
  • 설정
    • --gradio-img2img-tool = color-sketch (I2I 색 스케치)
    • Show progressbar = False
    • Stop At last layers of CLIP model = 2
    • Eta noise seed delta = 0 (없음)
  • 기타
    • 사용하는 모든 모델은 safetensors 임
    • 느그 나라 아이피만 접속할 수 있음
    • 모든 레포지토리는 항상 최신판 (스크립트 참고)
